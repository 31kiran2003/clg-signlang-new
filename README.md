## 🚀 Setup & Run
1. python -m venv myenv<br>
2. myenv\Scripts\activate<br>
3. pip install -r requirements.txt<br>
4. (Set path) Optional<br>
5. python .\main.py

## ▶️ To Run Later
1. myenv\Scripts\activate<br>
2. python .\main.py
